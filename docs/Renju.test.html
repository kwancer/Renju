(static) throwIfInvalid(gameState)

Source:
  • tests/renju.test.js, line 37

Returns if the board is in a valid state. A board is valid if all the following are true:

  • The board is a 2d array.
  • The board is a rectangular 2d array containing only null, "black", "white" or "removed".
  • At most one player has a winning configuration.
  • The moves are in order.
  • The center of the board has a black token if there are any moves.
Parameters:
Name Type Description
gameState gameState

The gameState to test.
Throws:

If the board fails any of the above conditions.
